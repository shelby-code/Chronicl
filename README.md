# Chronicl

An educational app to teach people about history.

## About

Chronicl makes learning history engaging and accessible. Explore historical events, figures, and timelines in an interactive format.

## Tech Stack

- [Vite](https://vite.dev/) + React + TypeScript
- [Tailwind CSS v4](https://tailwindcss.com/)

## Getting Started

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.
